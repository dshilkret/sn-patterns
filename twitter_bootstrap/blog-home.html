A Blog Home Template for Bootstrap 3

by Start Bootstrap

$[SP]Posted on August 28, 2013 at 10:00 PM

This is a very basic starter template for a blog homepage. It makes use of Glyphicons that are built into Bootstrap 3, and it makes use of the Pager at the bottom of the page. Make sure you get the Glyphicons files by downloading the entire /fonts$[SP]directory that you can download in the source files or at getbootstrap.com. That directory has all of the web fonts in it which makes Glyphicons work.
