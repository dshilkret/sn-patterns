Help with Start Bootstrap

Our help page is still under development. Please contact us if you have any issues or suggestions. Thanks!

Using the Templates

Each template is a standalone HTML file. Some templates also have a custom CSS document (ex. one-page-wonder.html$[SP]and one-page-wonder.css) You can either view the template preview and copy the HTML source code using your browser, or you can view the HTML on GitHub via the HTML button associated with each template. For the templates that have custom CSS styles attached to them, make sure you grab the corresponding CSS file either on GitHub or through your browser.

Still Having Trouble?

We will be extending this help page as we get add more templates and get more requests. If you have a specific questions, please email us at help@startbootstrap.com$[SP]and we will try our best to assist you.
